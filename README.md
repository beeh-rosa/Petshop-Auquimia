# Sistema Web - Auquimia

## Descrição do Projeto
Este projeto consiste na Fase 1 do desenvolvimento de uma plataforma online para um petshop, realizado para a disciplina de Fundamentos de Sistemas Web.

## Funcionalidades da Página Web
Como esta fase abrange apenas o desenvolvimento em HTML puro (sem CSS ou JavaScript), o sistema atua como uma vitrine estática e informativa. As principais funcionalidades incluem:

* **Navegação Semântica:** Uso de links no cabeçalho e rodapé para rolagem automática até as seções de Produtos e Serviços.
* **Vitrine de Produtos:** Exibição estruturada de três categorias de produtos (Acessórios, Rações Não Perecíveis e Higiene/Limpeza), contendo imagens, descrições detalhadas e preços.
* **Tabela de Serviços:** Listagem dos serviços de Banho e Tosa, oferecendo as modalidades com e sem tele-busca, com seus respectivos valores.
* **Informações Institucionais:** Rodapé contendo dados de autoria do projeto, direitos autorais e canais de contato com o petshop.

## Tecnologias Utilizadas
* HTML5 puro.
